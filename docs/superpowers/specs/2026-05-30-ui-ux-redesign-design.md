# UI/UX リデザイン仕様書

**日付:** 2026-05-30  
**対象:** `investment-app` — Next.js 16 投資管理アプリ  
**目的:** ダークモダンなダッシュボード型UIへの全面刷新。サイドバーナビゲーション・KPI重視の概要画面・自動バックグラウンド同期を導入し、PCユーザー向けのプロフェッショナルな体験を実現する。

---

## 前提条件

- **主要ターゲット:** PCブラウザ（レスポンシブ対応は不要、最小幅1024px想定）
- **デザインテーマ:** ダークモダン（Bloomberg/証券会社アプリ風）
- **AI機能:** フローティングFABボタン（右下固定、どの画面からも呼び出し可能）
- **同期自動化:** アプリ起動時に未同期銘柄を自動検知して自動実行（ユーザー操作不要）

## ページ1: 概要（デフォルト）

### KPIバー（ページ最上部）

4つの指標を横並びで表示：

| 指標 | 表示内容 |
|------|---------|
| 総評価額 | ¥XX,XXX,XXX（白・大字28px） |

| 損益率 | +X.XX%（グリーン or レッド・大字） |

### オンボーディングバナー（DB空 + 保有0件の場合のみ）

```
┌─────────────────────────────────────────────────────────────┐
│  ようこそ！まず銘柄を登録してみましょう                          │
│  1. 銘柄DBで日経225をワンクリック登録                          │
```

## AI フローティングボタン

**閉じているとき（全ページ共通・右下固定）:**

```
[💬 AIに相談]
```

**開いているとき（右下にパネル展開）:**

```
┌─────────────────────────────┐
│  AIアドバイザー        [×] │
│  ─────────────────────── │
│  （チャットメッセージ）       │
│                             │
│  ─────────────────────── │
│  [テキスト入力...]      [送信]│
└─────────────────────────────┘
```

- パネルサイズ: 幅380px × 高さ500px
- 位置: 右下固定（`position: fixed; bottom: 24px; right: 24px`）
- 開いていてもメインコンテンツは操作可能（モーダルではない）
- ポートフォリオデータを全ページで受け取り、どの画面からでも文脈を持って相談できる

## バックグラウンド同期

**トリガー:** アプリ起動時（`src/components/AutoSyncProvider.tsx` というクライアントコンポーネントで実行し、`layout.tsx` でラップする）

**ロジック:**
1. `GET /api/stocks/sync` を呼び `{ total: pendingCount }` を取得
2. `pendingCount > 0` であれば自動的に POST バッチを繰り返し実行
3. 同期中はヘッダーに `⟳ 同期中 X/Y` を表示
4. 完了したらインジケーターを消す
5. エラーが続いても静かに停止（ユーザーへのアラートなし）

**UI変更:**
- スクリーニング画面から「② Yahoo Finance から財務指標を同期」セクションを完全削除
- ヘッダーにグローバル同期ステータスインジケーターを追加
